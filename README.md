## Java Shopping Cart POO

Proyecto realizado en Java aplicando conceptos de Programación Orientada a Objetos mediante la simulación de un carrito de compras.

## Características

- Gestión de productos
- Agregado de productos al carrito
- Cálculo de totales
- Interacción entre objetos
- Encapsulación
- Uso de colecciones o arreglos

## Conceptos trabajados

- Clases y objetos
- Composición y asociación
- Métodos getters y setters
- Sobrescritura de métodos
- Manejo de colecciones
- Lógica de negocio

## Tecnologías utilizadas

- Java
- Eclipse IDE
